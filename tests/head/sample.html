[Free] Deploy Django project to PythonAnywhere.

Salah Ud Din

4 min read
·
3 days ago
< width="24" height="24" viewBox="0 0 24 24" aria-label="clap">

Complete tutorial.

  • Create Account on PythonAnywhere.
  • Update Settings.py.
  • Clone Project from GitHub.
  • Install requirements for project.
  • Create & Configure Web App on PythonAnywhere.
  • Create MySQL database.
  • Video Tutorial.

Create Account:

Go to Registration page : [Link], click on Create a beginner account.

After creating an account on PythonAnywhere .

Go to PythonAnywhere Dashboard click on console.

Click on Bash.

It will open the a console for you,

Before starting make few changes to your settings.py file.

Settings.py.

Add your app URL to ALLOWED_HOSTS <userName.pythonanywhere.com>
As you can see below i have added my app url just replace salah99 with your use name.

import dj_database_url

ALLOWED_HOSTS = ["127.0.0.1", "localhost", "salah99.pythonanywhere.com"]

# Replace it with your DATABASES.
DATABASES = {
    'default': dj_database_url.config(
        default="sqlite:///" + os.path.join(BASE_DIR, "db.sqlite3")
    )
}

When we create MySQL Database on PythonAnywhere it will load the the required credentials in DATABASE_URL (environment variable).

dj-database-url will load variables automatically from DATABASE_URL variable.

Clone Repo:

Now, open the GitHub repo and copy the .git URL.

Open console and clone the repository.

git clone <.git url of repo>

When you check the directory you will see you Project folder here it is “temp”

ls -al

Next, Most import step is to create the virtual environment.

python -m venv .venv

Now, Activate it.

source .venv/bin/activate

Now Install the required packages for your project:

Install requirements:

if you have requirements.txt file than, run command. pip install -r requirements.txt 

pip install dj-database-url
pip install django
pip install mysqlclient

wait for the installation to finish.

Create Web App:

Go to PythonAnywhere Dashboard.

and click next,

Select Django,

Select Python version,

Now, it will ask for Project name you can skip it. Because we don’t need it. as we are doing manual configuration.

let’s change it to webproject

Next.

Now go to Dashboard again, and click on “Web” tab.

And make two changes:

change project_home, where we cloned our project, “temp”. And change settings.py location, where your project settings file is.

Now, save it.

Update Virtual environment path, where you created the the venv.

Update static folder path as below, /home/userName/ProjectFolder/StaticFolderName

Database Setup:

App setup is down now open Dashboard again and go to Databases.

click on MySQL, It will ask for password enter password and Done.

Go to bash again,

python manage.py collectstatic

Now, make migrate.

python manage.py makemigrations
python manage.py migrate

Everything is Done, Now reload the App, Enjoy.
